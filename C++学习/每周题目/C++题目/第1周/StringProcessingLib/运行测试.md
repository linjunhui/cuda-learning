# 如何编译和运行测试

## 编译步骤

```bash
# 1. 进入项目目录
cd 每周题目/C++题目/第1周/StringProcessingLib

# 2. 创建构建目录
mkdir -p build
cd build

# 3. 配置 CMake（会自动下载 Google Test）
cmake ..

# 4. 编译
make
# 或者使用多核编译
make -j4
```

## 运行测试

### 方式 1：直接运行（推荐，查看详细输出）

```bash
cd build
./test_fixed_size_pool
```

### 方式 2：使用 CTest（集成测试）

```bash
cd build
ctest --output-on-failure
```

### 方式 3：运行特定测试

```bash
cd build
# 只运行 Initialization 测试
./test_fixed_size_pool --gtest_filter=FixedSizePoolTest.Initialization

# 运行所有 FixedSizePoolTest 的测试
./test_fixed_size_pool --gtest_filter=FixedSizePoolTest.*
```

### 方式 4：生成 XML 测试报告

```bash
cd build
./test_fixed_size_pool --gtest_output=xml:test_results.xml
```

## 常见问题

### 如果测试失败，查看详细信息：

```bash
./test_fixed_size_pool --gtest_brief=0  # 显示所有详情
```

### 如果需要调试测试：

```bash
# 编译 Debug 版本
cmake -DCMAKE_BUILD_TYPE=Debug ..
make

# 使用 gdb 调试
gdb ./test_fixed_size_pool
```
