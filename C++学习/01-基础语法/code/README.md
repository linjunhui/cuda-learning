## 编译和运行

### 编译单个文件

```bash
# 编译题目1
g++ -std=c++11 -o week01_problem01 week01_problem01.cpp

# 运行
./week01_problem01
```

### 编译所有文件

```bash
# 在src目录下
cd src
for file in week01_*.cpp; do
    g++ -std=c++11 -o "${file%.cpp}" "$file"
done
```

### 使用CMake（可选）

如果需要使用CMake，可以创建CMakeLists.txt文件。

## 学习建议

1. **按顺序学习**：建议按照题目编号顺序完成，因为后面的题目会用到前面的知识点。

2. **理解注释**：每个文件开头都有详细的题目描述、要求、知识点说明，请仔细阅读。

3. **独立完成**：先尝试自己实现，遇到困难再参考代码。

4. **修改和扩展**：完成基本要求后，可以尝试修改和扩展功能。

5. **调试练习**：遇到错误时，学会使用调试工具（gdb）来定位问题。

## 编译选项说明

- `-std=c++11`：使用C++11标准
- `-Wall`：显示所有警告
- `-g`：包含调试信息
- `-O2`：优化级别2

示例：
```bash
g++ -std=c++11 -Wall -g -O2 -o week01_problem01 week01_problem01.cpp
```

## 注意事项

1. 所有代码都使用C++11标准编写
2. 代码中包含异常处理，注意理解异常的使用
3. 部分代码使用了STL容器和算法，需要理解其用法
4. 项目1涉及模板编程，需要理解类模板的概念
