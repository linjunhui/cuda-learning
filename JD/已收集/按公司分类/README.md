# 按公司分类收集

## 分类说明

本目录按照公司类型和规模对收集的工作描述进行分类整理，便于了解不同公司的招聘特点和要求。

## 分类体系

### 按公司类型分类

#### 互联网公司
- **一线互联网**：腾讯、阿里巴巴、字节跳动、百度、美团、滴滴等
- **二线互联网**：网易、京东、拼多多、快手、小米、360等
- **新兴互联网**：新兴的互联网公司和独角兽企业

#### 外企公司
- **科技外企**：Google、Microsoft、Apple、Amazon、NVIDIA、Intel等
- **咨询外企**：IBM、Oracle、SAP等
- **金融外企**：高盛、摩根大通、花旗等

#### 传统企业
- **制造业**：华为、中兴、比亚迪、吉利等
- **金融业**：工商银行、建设银行、招商银行等
- **能源业**：国家电网、中石油、中石化等

#### 创业公司
- **早期创业**：A轮及以前的公司
- **成长期创业**：B轮、C轮的公司
- **成熟期创业**：D轮及以后的公司

### 按公司规模分类

#### 大型企业（1000人以上）
- **超大型**：万人以上
- **大型**：1000-10000人

#### 中型企业（100-1000人）
- **中大型**：500-1000人
- **中型**：100-500人

#### 小型企业（100人以下）
- **小型**：50-100人
- **微型**：50人以下

## 文件命名规范

### 文件命名格式
```
[公司类型]_[公司名称]_[职位名称]_[发布时间].md
```

### 示例
```

创业公司_某AI公司_深度学习工程师_2024-01-13.md
```

## 公司特点分析

### 互联网公司特点
- **技术导向**：重视技术创新和前沿技术应用
- **快速迭代**：产品更新迭代速度快
- **年轻化**：团队年轻，工作氛围活跃
- **高薪**：薪资水平相对较高
- **工作强度**：工作强度较大，加班较多

### 外企公司特点
- **国际化**：工作环境国际化，英语要求高
- **规范化**：管理制度规范，流程完善
- **福利好**：福利待遇较好，工作生活平衡
- **技术先进**：技术相对先进，学习机会多
- **稳定性**：工作相对稳定，裁员风险较低

### 传统企业特点
- **稳定**：工作稳定性好，裁员风险低
- **福利完善**：福利制度完善，保障较好
- **晋升缓慢**：晋升速度相对较慢
- **技术保守**：技术相对保守，创新不足
- **工作轻松**：工作强度相对较小

### 创业公司特点
- **机会多**：发展机会多，成长空间大
- **风险高**：公司风险较高，稳定性差
- **学习快**：学习机会多，技能提升快
- **工作强度大**：工作强度大，压力较大
- **股权激励**：可能有股权激励机会

## 收集建议

### 1. 重点关注公司
- **目标公司**：根据个人职业规划选择重点关注的公司
- **标杆公司**：选择行业标杆公司作为参考标准
- **新兴公司**：关注新兴的潜力公司

### 2. 收集策略
- **全面收集**：对重点公司进行全面收集
- **定期更新**：定期更新已收集公司的信息
- **深度分析**：对重点公司进行深度分析

### 3. 信息验证
- **多渠道验证**：通过多个渠道验证信息准确性
- **时效性检查**：定期检查信息的时效性
- **交叉对比**：与其他公司进行交叉对比分析

## 使用建议

### 1. 求职策略
- **目标选择**：根据个人情况选择合适的目标公司
- **准备策略**：针对不同公司制定不同的准备策略
- **面试准备**：了解不同公司的面试特点和要求

### 2. 职业规划
- **发展方向**：根据公司特点规划职业发展方向
- **技能发展**：根据公司要求发展相应技能
- **长期规划**：制定长期的职业发展规划

### 3. 学习指导
- **技术学习**：根据目标公司的技术要求进行学习
- **能力提升**：提升目标公司重视的能力
- **经验积累**：积累目标公司看重的经验
