# 设计阶段学习资料

## 概述

设计阶段是HPPD开发过程的重要环节，主要关注方案设计和设计评估能力。

## 能力清单

1. **设计评估** - 能够参与设计评审并提出改进建议，识别可测试性需求

## 学习路径

### 基础阶段（1级 - 0.5分）
- 参与特性或功能的设计评审
- 给出有效评审意见

### 进阶阶段（2级 - 1分）
- 对特性或功能相关方案提出改进建议并被采纳

### 高级阶段（3级 - 1.5分）
- 现场部署或应用场景识别
- 提出可测试性需求并被采纳

## 实践建议

1. 积极参与设计评审活动
2. 学习测试性设计原则
3. 从测试角度思考设计方案
4. 提出可测试性改进建议
