# 测试方案/计划

## 能力清单

### 测试方案

#### 1级（0.5分）
- 输出需求实验室测试方案

#### 2级（1分）
- 根据需求和领域要求，输出领域实验室测试方案

#### 3级（1.5分）
- 根据需求和产品要求，输出实验室版本测试方案

### 外场测试方案

#### 1级（0.5分）
- 输出需求外场测试方案

#### 2级（1分）
- 根据领域和外场要求，输出领域外场测试方案

#### 3级（1.5分）
- 根据产品和外场要求，输出产品外场测试方案

### 对外交流能力

#### 1级（0.5分）
- 独立负责外场功能的测试

#### 2级（1分）
- 独立负责外场功能宣讲、测试和引导

#### 3级（1.5分）
- 能够独立完成与客户的交流和汇报工作，对客户的问题能做到流利的解答和处理

## 学习目标

- [ ] 掌握测试方案的编写方法
- [ ] 理解需求级、领域级、产品级测试方案的区别
- [ ] 掌握外场测试方案的编写
- [ ] 提升对外交流能力
