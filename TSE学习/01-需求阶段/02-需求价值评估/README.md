# 需求价值评估

## 能力要求

### 1级（0.5分）
- 能对需求本身描述中的价值进行正确性验证

### 2级（1分）
- 站在用户的角度进行思考，能对需求的价值的正确性进行验证

### 3级（2分）
- 对需求的价值的正确性进行验证，并且能提出新的需求价值点，增加需求的完备性

## 学习目标

- [ ] 掌握需求价值验证方法
- [ ] 从用户角度思考需求价值
- [ ] 实践需求价值验证
- [ ] 能够提出新的需求价值点

## 学习资源

### 📚 理论知识

#### 核心概念
1. **需求价值是什么**
   - 用户价值：用户能获得什么好处
   - 业务价值：业务能获得什么收益
   - 技术价值：技术上有何改进

2. **价值评估方法**
   - 价值验证方法：检查需求描述中的价值是否合理
   - 用户角度思考：从用户视角分析价值
   - 价值创新：提出新的价值点

3. **价值判断标准**
   - 价值是否明确？
   - 价值是否可衡量？
   - 价值是否合理？
   - 价值能否实现？
