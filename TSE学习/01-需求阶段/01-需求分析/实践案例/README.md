## 案例编写指南

### 案例应包含的内容

1. **基本信息**
   - 案例名称、类型、应用能力、时间等

2. **案例背景**
   - 项目/任务描述
   - 目标和期望

3. **实践过程**
   - 准备阶段
   - 执行阶段（详细步骤）
   - 总结阶段

4. **实践结果**
   - 成果
   - 数据/证据
   - 评估

5. **能力体现**
   - 应用的能力项
   - 能力提升情况

6. **经验总结**
   - 成功经验
   - 遇到的问题
   - 改进建议

7. **心得体会**

8. **后续计划**

### 案例质量要求

- **真实性**：基于真实项目（可脱敏）
- **完整性**：包含完整的实践过程
- **可学习性**：其他学员可以从案例中学习
- **可复用性**：经验和方法可以应用到其他场景

## 贡献案例

## 学习建议

1. **按难度顺序学习**：先学习简单案例，再学习复杂案例
2. **对比学习**：对比不同案例的处理方法
3. **实践应用**：学习案例后，在实际项目中应用
4. **总结反思**：记录自己的实践经验和改进
