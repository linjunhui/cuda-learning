# 大模型学习资料

## 概述

大模型能力是HPPD开发过程中的新兴能力，主要关注AI工具使用、提示词工程、知识库使用、AI评测等方面的能力。

## 能力清单

### 通用大模型能力
1. **AI工具使用能力** - 能够使用AI工具进行业务活动，优化AI应用范式
2. **提示词工程能力** - 能够编写有效的Prompt引导大模型完成工作
3. **知识库使用能力** - 能够提供训练语料，参与知识工程流水线
4. **AI评测** - 能够提供评测语料，执行评测流水线
5. **模型训练精调** - 了解模型训练和精调的基本知识
6. **AI应用测试评估** - 具备AI在测试域应用的效能分析能力
7. **AI应用范式探索** - 探索AI新技术，分析和补充测试域的AI应用场景

## 学习路径

### 基础阶段（1级）
- 使用AI工具进行业务活动
- 编写基础的Prompt
- 提供训练和评测语料
- 了解模型训练和精调知识

### 进阶阶段（2级）
- 熟练掌握AI工具，优化应用范式
- 使用高级提示词策略
- 参与知识工程流水线
- 执行评测流水线和数据分析

### 高级阶段（3级）
- 设计测试域综合智能体
- 构建和维护测试域知识体系
- 设计评测集并解读数据
- 探索AI应用范式
