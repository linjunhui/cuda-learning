## 📚 学习内容

### AI Agent 核心架构

**定义**：能自主感知环境、规划并执行任务的智能系统

**核心组件**：
- 🧠 **大脑 LLM**：大语言模型作为核心决策引擎
- 👁️ **感知**：环境感知和信息获取能力
- ⚡ **执行**：任务执行和动作输出
- 💾 **记忆**：信息存储和知识管理

**与传统自动化的区别**：
- ✅ 基于目标自主调整策略
- ✅ 能处理模糊任务
- ❌ 传统自动化依赖预设规则

### 三项关键技术

1. **MCP (Model Context Protocol)**
   - 核心作用：连接 LLM 与外部工具
   - 工作流程：定义标准化的工具调用协议
   - 本质价值：提升 Agent 的实用性和可扩展性

2. **Function Calling**
   - LLM 调用外部函数的机制
   - 实现 LLM 与外部系统的集成

3. **A2A (Agent to Agent)**
   - 多 Agent 协作通信机制
   - 实现复杂的分布式任务处理

### RAG（检索增强生成）

**定义**：Retrieval Augmented Generation

**核心思想**：
- 结合检索和生成技术
- 从知识库中检索相关信息
- 基于检索结果生成更准确的回答
