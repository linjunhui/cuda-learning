# 需求阶段学习资料

## 概述

需求阶段是HPPD开发过程的起始阶段，主要包括需求分析、需求价值评估和需求场景评估三个核心能力。

## 能力清单

1. **需求分析方法** - 掌握需求分析基础知识和实例化需求方法
2. **需求价值评估** - 能够验证需求价值的正确性和完备性
3. **需求场景评估** - 能够验证需求场景的正确性和完备性

## 学习路径

### 基础阶段（1级）
- 了解需求分析的基础知识
- 了解实例化需求概念和优点
- 能对需求价值和场景进行基本验证

### 进阶阶段（2级）
- 熟悉实例化需求的关键过程模式
- 站在用户角度思考需求价值
- 能发现BA/SE未考虑到的场景

### 高级阶段（3级）
- 能提出新的需求价值点，增加需求完备性
