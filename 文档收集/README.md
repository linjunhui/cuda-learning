# cuda-learning

> 系统性的 C++ 和 CUDA 编程学习工程，面向 C++ 软件研发工程师（CUDA 方向）

[![Status](https://img.shields.io/badge/status-active-success.svg)]()

## 📖 项目简介

`cuda-learning` 是一个系统性的 C++ 和 CUDA 编程学习工程，采用理论与实践相结合的学习方式。项目持续更新中，旨在培养面向 C++ 软件研发工程师（CUDA 方向）的技术能力。

### 核心特点

- ✅ **系统性学习**：C++ 和 CUDA 各分为 7 个学习阶段，循序渐进
- ✅ **项目实践**：每周题目提供真实项目场景，积累实战经验
- ✅ **面试准备**：包含 35+ 道内存池设计面试题，覆盖各个难度级别
- ✅ **知识库汇总**：详细的知识库文档，包含技术细节和代码示例
- ✅ **JD 分析**：收集和分析目标职位要求，针对性学习

## 🚀 快速开始

### 1. 克隆仓库

```bash
git clone https://github.com/linjunhui/cuda-learning.git
cd cuda-learning
```

### 2. 环境要求

- **C++ 编译器**：GCC 9+ / Clang 10+ / MSVC 2019+
- **CUDA Toolkit**：CUDA 11.0+
- **构建工具**：CMake 3.15+
- **测试框架**：Google Test（自动下载）

### 3. 构建第一个项目

```bash
# 进入第1周项目目录
cd 每周题目/C++题目/第1周/StringProcessingLib

# 创建构建目录
mkdir build && cd build

# 配置和构建
cmake ..
make

# 运行测试
ctest
```

### 4. 查看学习计划

详细的学习计划请参考：[学习计划执行指南.md](学习计划执行指南.md)

## 📚 学习路径

### C++ 学习路径

1. **基础语法**（1-2个月）
   - 数据类型、变量、函数
   - 控制流、数组、指针
   - 内存管理、智能指针

2. **面向对象**（1-2个月）
   - 类与对象、继承与多态
   - 抽象类、接口设计
   - 设计模式基础

3. **数据结构算法**（1-2个月）
   - STL 容器和算法
   - 常用数据结构实现
   - 算法复杂度分析

4. **高级特性**（2-3个月）
   - 模板编程、元编程
   - 现代 C++（C++11/14/17/20）
   - 类型推导、Lambda 表达式

5. **并发编程**（1-2个月）
   - 多线程编程
   - 线程同步、无锁编程
   - 异步编程、协程

6. **性能优化**（1-2个月）
   - 性能分析工具
   - 内存优化、CPU 优化
   - 编译器优化技巧

7. **项目实践**（持续）
   - 综合项目开发
   - 代码审查、重构
   - 工程化实践

### CUDA 学习路径

1. **基础概念**（1个月）
   - CUDA 编程模型
   - 线程组织、内存模型
   - 基础 Kernel 编写

2. **内存管理**（1个月）
   - 全局内存、共享内存
   - 常量内存、纹理内存
   - 内存优化策略

3. **性能优化**（2-3个月）
   - 性能分析工具（Nsight）
   - 内存访问优化
   - 占用率优化

4. **CUDA 库工具**（1-2个月）
   - cuBLAS、cuDNN
   - Thrust、cuFFT
   - CUDA 运行时 API

5. **GPU 工程应用**（2-3个月）
   - 图像处理、深度学习
   - 科学计算、并行算法
   - 实际项目开发

6. **分布式计算**（1-2个月）
   - 多 GPU 编程
   - 集群计算
   - 通信优化

7. **项目实践**（持续）
   - CUDA 项目开发
   - 性能调优实践
   - 工程化部署

### 对于初学者

1. **按顺序学习**：先完成 C++ 基础语法，再开始 CUDA
