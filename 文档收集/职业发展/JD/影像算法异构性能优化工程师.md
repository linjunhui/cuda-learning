# 影像算法异构性能优化工程师

## 职位概述
负责影像算法的异构性能优化、端侧工程化部署、项目交付等工作，构建影像算法在性能、功耗领域的竞争力，打造极致用户体验。

## 工作职责

### 1. 影像算法性能优化
- 负责影像算法的异构性能优化
- 端侧工程化部署
- 项目交付管理
- 构建影像算法在性能、功耗领域的竞争力
- 打造极致用户体验

### 2. 平台能力优化
- 基于QCom/MTK等平台能力进行算子模块优化
- 算法移植工作
- 并行计算加速
- 功耗调优

### 3. 异构计算单元开发
- 针对影像算法需求，对主SoC、外挂芯片等的异构计算单元XPU进行分析
- 规格分析和需求设计
- 解决方案开发
- 验证测试

## 任职要求

### 1. 学历要求
- 计算机、电子工程相关专业
- 研究生及以上学历
- 3年以上算法部署相关经验

### 2. 编程技能
- 熟练使用C/C++开发语言
- 良好的编程习惯
- 良好的文档编写能力

### 3. 优化技术
- 熟练使用arm neon intrinsics
- 熟练使用OpenCL/OpenGL/Vulkan
- 熟练使用xtensa/Hexagon DSP
- 掌握任意一种或多种优化手段

### 4. 架构知识
- 熟悉异构计算架构（如GPU/CPU/NPU）
- 熟悉相关Profiling工具
- 具备算法部署的分析能力

### 5. 算法经验
- 熟悉CV图像算法
- 做过算法性能内存功耗优化相关工作者优先

### 6. 框架经验
- 了解基于移动端的AI前向推理框架
- 熟悉NCNN、MNN、QNN等框架
- 有相关开发经验者优先

### 技术面试
1. **C/C++基础**: 指针、内存管理、STL等
2. **算法优化**: ARM NEON、OpenCL等优化技术
3. **图像处理**: 常见CV算法原理和实现
4. **系统架构**: 异构计算、内存管理、性能分析

### 项目经验
1. 准备2-3个相关的项目经验
2. 重点描述性能优化的过程和结果
3. 准备技术难点和解决方案
4. 量化优化效果（性能提升、功耗降低等）

### 学习建议
1. **理论学习**: 深入学习计算机视觉和图像处理算法
2. **实践项目**: 参与开源项目或自己实现算法
3. **优化技术**: 学习ARM NEON、OpenCL等优化技术
4. **工具使用**: 熟悉Profiling工具和性能分析方法
