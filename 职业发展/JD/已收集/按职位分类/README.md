# 按职位分类收集

## 分类说明

本目录按照职位类型对收集的工作描述进行分类整理，便于快速查找和分析特定类型的职位。

## 分类体系

### 技术开发类
- **CUDA开发工程师**：专注于CUDA编程和GPU计算
- **GPU算法工程师**：GPU算法设计和优化
- **高性能计算工程师**：HPC系统开发和优化
- **图形学工程师**：计算机图形学和渲染技术
- **AI算法工程师**：人工智能算法开发
- **机器学习工程师**：机器学习模型开发
- **深度学习工程师**：深度学习框架和模型
- **计算机视觉工程师**：图像处理和视觉算法

### 系统架构类
- **系统架构师**：系统整体架构设计
- **分布式系统工程师**：分布式系统开发
- **云平台工程师**：云计算平台开发
- **DevOps工程师**：开发和运维一体化

### 数据科学类
- **数据科学家**：数据分析和建模
- **数据工程师**：数据处理和管道构建
- **大数据工程师**：大数据平台开发

### 产品管理类
- **技术产品经理**：技术产品规划和设计
- **AI产品经理**：AI产品管理和推广

### 研究类
- **算法研究员**：前沿算法研究
- **技术研究员**：技术前沿探索
- **博士后研究员**：学术研究

## 文件命名规范

### 文件命名格式
```
[职位类型]_[公司名称]_[发布时间].md
```

### 示例
```
CUDA开发工程师_英伟达_2024-01-15.md

```

## 收集指南

### 1. 职位分类标准
- **主要职责**：根据主要工作职责进行分类
- **技能要求**：根据核心技能要求进行分类
- **行业特点**：根据所在行业特点进行分类

### 2. 跨类别职位处理
- 如果职位涉及多个类别，选择主要类别
- 在文件内容中标注其他相关类别
- 创建交叉引用链接

### 3. 细分分类
- 每个大类下可以进一步细分
- 根据实际收集情况进行调整
- 保持分类体系的一致性
