# JD (Job Description) 工作描述收集

## 目录说明

本目录用于收集和分析各种招聘软件上的工作描述，帮助了解市场需求和职位要求。


## 收集指南

### 1. 收集原则
- **完整性**：收集完整的职位描述，包括职责、要求、薪资等信息
- **时效性**：标注收集日期，定期更新过时信息
- **准确性**：确保信息准确无误，避免误导
- **分类性**：按照统一的分类标准进行整理

### 2. 收集渠道
- **主流招聘网站**：
  - 智联招聘 (zhaopin.com)
  - 前程无忧 (51job.com)
  - Boss直聘 (zhipin.com)
  - 拉勾网 (lagou.com)
  - 猎聘网 (liepin.com)
- **技术社区**：
  - GitHub Jobs
  - Stack Overflow Jobs
  - V2EX 招聘版
- **公司官网**：
  - 各大互联网公司招聘页面
  - 外企招聘信息

### 3. 收集内容
- **基本信息**：
  - 公司名称
  - 职位标题
  - 工作地点
  - 薪资范围
  - 发布时间
- **职位描述**：
  - 工作职责
  - 任职要求
  - 技能要求
  - 经验要求
  - 学历要求
- **其他信息**：
  - 福利待遇
  - 公司介绍
  - 联系方式

## 分析方法

### 1. 技能需求分析
- 统计高频技能关键词
- 分析技能组合模式
- 识别新兴技能趋势

### 2. 薪资水平分析
- 按地区分析薪资分布
- 按经验分析薪资差异
- 按技能分析薪资溢价

### 3. 市场需求分析
- 职位数量变化趋势
- 不同领域需求对比
- 季节性招聘规律
